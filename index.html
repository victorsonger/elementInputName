推荐验证步骤

  1. 用 HTTP 服务打开页面，分别填写每个卡片中的输入框并提交。
  2. 刷新页面后再次聚焦相同输入框，观察是否出现浏览器历史输入建议或密码管理器弹窗。
  3. 重点对比 autocomplete="off" + 无稳定 name 的手机号输入，与 name="phone" autocomplete="tel" 或 name="username" autocomplete="username" 的差异。
  4. 账号密码表单更可能触发密码管理器；手机号验证码表单通常只依赖普通表单历史记录。
